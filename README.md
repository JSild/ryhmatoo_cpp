# C++ rühmatöö
## Rühm AD: Convery's Game of Life

Meie rühmatöö eesmärk on luua Conway elumängu simulaator. Conway elumäng on
rakuautomaat, mille lõi 1970. aastal matemaatik John Horton Conway. See on nullmängijaga
mäng, mis tähendab, et pärast algseisu määramist areneb süsteem edasi kindlate reeglite
järgi ilma kasutaja täiendava sekkumiseta. Mängu käik toimub põlvkondade kaupa ning
sellel puudub kindel lõpp või võidutingimus.

Programm kuvab ruudustiku, mille igas lahtris võib olla elus või surnud rakk. Kasutaja saab
määrata algseisu, simulatsiooni käivitada ja peatada ning soovi korral põlvkonniti edasi
liikuda. Tahame ka lisada võimaluse muuta hiirega rakkude olekut, lähtestada algseisu ning
kuvada hetke põlvkonna numbrit. Võimalusel lisame ka juhusliku algseisu loomise ja mõned
eeldefineeritud mustrid.

Programmi kompileerimiseks kasutame Makefile’i ja graafilise kasutajaliidese jaoks plaanime
kasutada SFML teeki.
